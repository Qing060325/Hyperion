# Hyperion v0.3.0 更新日志

## 🎉 新功能

### 一键安装包、傻瓜式配置 (P0)

- ✨ **欢迎向导** - 首次启动时显示多步骤引导
  - 自动检测本地 Clash 内核
  - 一键配置连接参数
  - 支持跳过向导

- 🚀 **开箱即用体验**
  - 自动检测运行中的 Clash 实例
  - 智能推荐配置
  - 连接测试一键验证

### 完整可视化规则编辑器 (P1)

- 📝 **规则编辑器**
  - 可视化规则类型选择器
  - 表单式规则编辑
  - 支持规则的增删改查
  - 拖拽排序规则

- ✅ **规则验证**
  - 实时验证规则格式
  - 错误提示和警告
  - 预览 YAML 格式输出

- 📦 **批量操作**
  - 导入规则文件
  - 导出规则配置
  - 复制/删除规则

### 策略组拖拽 (P1)

- 🎯 **可视化排序**
  - 使用 @dnd-kit 实现拖拽
  - 支持触摸设备
  - 拖拽时视觉反馈

- 💾 **排序持久化**
  - 保存排序到配置
  - 撤销修改
  - 批量更新策略组

### 多配置文件管理 (P1)

- 📁 **配置文件列表**
  - 显示所有配置文件
  - 本地/远程配置标识
  - 快速切换配置

- 📥 **导入导出**
  - 导入本地 YAML 文件
  - 从 URL 导入远程配置
  - 导出配置文件

- 🔄 **远程配置更新**
  - 一键更新远程配置
  - 查看更新时间
  - 配置重命名/删除

### TUN/系统代理模式切换 (P2)

- 🌐 **TUN 模式**
  - 开关切换
  - 状态显示
  - 配置信息展示

- 🔧 **系统代理**
  - 一键设置系统代理
  - 端口配置显示
  - 高级设置入口

### 连接详情增强 (P2)

- 🗺️ **IP 地理定位**
  - 自动查询 IP 地理位置
  - 显示国旗和城市
  - ISP 信息展示

- 📊 **连接详情面板**
  - 完整连接信息
  - 代理链路可视化
  - 流量统计
  - 主动断开连接

### 日志过滤增强 (P2)

- 🔍 **高级过滤**
  - 多条件组合过滤
  - 关键词高亮
  - 时间范围筛选

- 📤 **日志导出**
  - TXT/JSON/CSV 格式
  - 自定义导出范围
  - 日志统计

### 插件/主题扩展 (P3)

- 🎨 **主题系统**
  - 内置多款主题
  - 深色/浅色模式
  - 跟随系统设置

- 🔌 **插件架构**
  - 插件管理器
  - 扩展 API
  - 生命周期钩子

### 快捷键与托盘集成 (P3)

- ⌨️ **全局快捷键**
  - 可自定义快捷键
  - 预设常用操作
  - 启用/禁用控制

- 🔔 **通知系统**
  - 连接错误通知
  - 订阅到期提醒
  - 新版本通知
