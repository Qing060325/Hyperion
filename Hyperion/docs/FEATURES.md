## 快速开始

### 环境要求

| 依赖 | 最低版本 |
|------|----------|
| Node.js | 18+ |
| pnpm | 8+ |
| Clash Meta | 最新版 |

### 安装与运行

```bash
# 克隆仓库
git clone https://github.com/Qing060325/Hyperion.git
cd Hyperion

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

### 首次运行

首次启动时，Hyperion 会显示欢迎向导，帮助你：

1. 自动检测本地 Clash 内核
2. 配置 API 连接参数
3. 完成初始化设置

## 规则编辑器

### 访问路径

导航至 **规则** → **编辑规则** 或访问 `/rules/editor`

### 支持的规则类型

#### 域名规则

| 类型 | 说明 | 示例 |
|------|------|------|
| DOMAIN | 完整域名匹配 | `DOMAIN,google.com,Proxy` |
| DOMAIN-SUFFIX | 域名后缀匹配 | `DOMAIN-SUFFIX,google.com,Proxy` |
| DOMAIN-KEYWORD | 域名关键词匹配 | `DOMAIN-KEYWORD,google,Proxy` |
| DOMAIN-REGEX | 正则表达式匹配 | `DOMAIN-REGEX,^.*google.*$,Proxy` |

#### IP 规则

| 类型 | 说明 | 示例 |
|------|------|------|
| IP-CIDR | IPv4 CIDR 匹配 | `IP-CIDR,192.168.1.0/24,Direct` |
| IP-CIDR6 | IPv6 CIDR 匹配 | `IP-CIDR6,2001:db8::/32,Direct` |
| IP-SUFFIX | IP 后缀匹配 | `IP-SUFFIX,8.8.8.8,Proxy` |

#### 地理位置规则

| 类型 | 说明 | 示例 |
|------|------|------|
| GEOIP | IP 地理位置 | `GEOIP,CN,Direct` |
| GEOSITE | 网站地理位置 | `GEOSITE,google,Proxy` |
| SRC-GEOIP | 源 IP 地理位置 | `SRC-GEOIP,CN,Direct` |

#### 进程规则

| 类型 | 说明 | 示例 |
|------|------|------|
| PROCESS-NAME | 进程名匹配 | `PROCESS-NAME,chrome.exe,Proxy` |
| PROCESS-PATH | 进程路径匹配 | `PROCESS-PATH,/usr/bin/curl,Proxy` |

#### 逻辑规则

| 类型 | 说明 | 示例 |
|------|------|------|
| AND | 多条件与 | `AND,((DOMAIN,x.com),(DST-PORT,443)),Proxy` |
| OR | 多条件或 | `OR,((DOMAIN,a.com),(DOMAIN,b.com)),Proxy` |
| NOT | 条件取反 | `NOT,((GEOIP,CN)),Proxy` |

### 使用方法

1. **添加规则**：点击"添加规则"按钮
2. **选择类型**：从下拉菜单选择规则类型
3. **填写内容**：输入匹配内容和目标策略
4. **保存规则**：点击"保存"按钮

### 批量操作

- **导入**：从 YAML 文件导入规则
- **导出**：导出为 YAML 格式
- **复制**：快速复制现有规则
- **删除**：批量删除选中规则

## 策略组拖拽

### 启用排序模式

1. 进入代理页面
2. 点击"排序模式"按钮
3. 拖拽节点卡片调整顺序
4. 点击"保存排序"确认

### 注意事项

- 排序会修改配置文件
- 建议在排序前备份配置
- 部分代理组（如 URLTest）不支持手动排序

## 多配置文件管理

### 访问路径

导航至 **配置文件** 或访问 `/profiles`

### 导入配置

#### 本地文件导入

1. 点击"添加本地配置"
2. 选择 YAML 格式的配置文件
3. 或直接拖拽文件到导入区域

#### URL 导入

1. 点击"从 URL 导入"
2. 输入订阅链接
3. 可选填写名称
4. 点击"导入"

### 配置操作

| 操作 | 说明 |
|------|------|
| 切换 | 点击配置卡片切换到该配置 |
| 重命名 | 点击菜单 → 重命名 |
| 导出 | 导出为 YAML 文件 |
| 更新 | 更新远程配置（仅远程配置） |
| 删除 | 删除配置文件（不能删除当前使用的配置） |

## 网络模式切换

### TUN 模式

TUN 模式提供系统级代理能力。

#### 启用方式

1. 进入设置 → 网络
2. 打开"TUN 模式"开关
3. 确认需要管理员权限

#### 配置说明

| 配置项 | 说明 |
|--------|------|
| Stack | 协议栈：gvisor / system / lwip |
| 自动路由 | 自动配置系统路由表 |
| DNS 劫持 | 劫持 DNS 请求 |

### 系统代理

系统代理通过修改系统代理设置实现。

#### 支持的代理类型

- HTTP 代理
- HTTPS 代理
- SOCKS5 代理

## 连接详情

### 查看详情

点击连接列表中的连接条目即可查看详情面板。

### 显示信息

| 信息 | 说明 |
|------|------|
| 目标主机 | 请求的目标地址和端口 |
| 代理链路 | 完整的代理转发路径 |
| 地理位置 | 目标 IP 的地理位置 |
| 进程信息 | 发起请求的进程名 |
| 匹配规则 | 命中的规则信息 |
| 流量统计 | 上传/下载流量 |
| 连接时间 | 连接建立时间和持续时长 |

### IP 地理定位

使用 ip-api.com 免费服务查询 IP 地理位置，支持：

- 国家/地区
- 省份/城市
- ISP 信息
- 国旗显示

## 日志过滤

### 访问路径

进入日志页面，展开过滤面板。

### 过滤条件

| 条件 | 说明 |
|------|------|
| 级别 | Debug / Info / Warning / Error |
| 关键词 | 支持多个关键词 |
| 时间范围 | 开始和结束时间 |
| 进程 | 按进程名过滤 |
| 排除 | 排除特定关键词 |

### 预设过滤器

| 预设 | 说明 |
|------|------|
| 全部 | 显示所有日志 |
| 错误 | 仅显示 Error 级别 |
| 警告及以上 | Warning 和 Error |
| 重要 | Info 及以上 |

### 导出日志

支持导出为以下格式：

- **TXT** - 纯文本格式
- **JSON** - 结构化数据
- **CSV** - 表格格式

## API 扩展

### RESTful API

Hyperion 通过 Clash RESTful API 与内核通信。

#### 常用端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /version | 获取版本信息 |
| GET | /configs | 获取当前配置 |
| PATCH | /configs | 更新配置 |
| GET | /proxies | 获取代理列表 |
| PUT | /proxies/{name} | 切换代理 |
| GET | /rules | 获取规则列表 |
| GET | /connections | 获取连接列表 |
| DELETE | /connections/{id} | 关闭连接 |

### WebSocket 流

| 端点 | 数据类型 |
|------|----------|
| /traffic | 实时流量 |
| /logs | 实时日志 |
| /connections | 实时连接 |
